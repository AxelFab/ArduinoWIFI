Contolar LED con arduino por medio de WIFI

Componentes requeridos:

  • Arduino UNO
  • Módulo Wi-Fi ESP8266
  • Cable USB
  • Conexión de cables
  • LED RGB
  • Teléfono móvil Android
  • Aplicación Blynk



Paso para configurar la aplicación Blynk:

  • 1. Primero, descárguelo de Google Play Store e instálelo en un teléfono móvil Android.
  • 2. Después de esto, se requiere crear una cuenta. Puede usar su cuenta actual de Gmail.
  • 3. Ahora selecciona Arduino Board y dale un nombre para tu proyecto.
  • 4. Anote el Auth Token Code o simplemente envíelo por correo a su cuenta de correo electrónico y luego cópielo y péguelo en Arduino sketch (Código de programa).
  • 5. Ingresa este Código de autenticación de Token en el boceto de Arduino. // Deberías obtener Auth Token en la aplicación Blynk. // Vaya a la configuración del proyecto (icono de la tuerca).
    char auth [] = "a20b235cfa794f07981d050950fb4429";
  • 6. Luego haz clic en el botón Crear en la aplicación Blynk.
  • 7. Ahora seleccione el control deslizante grande y dos botones, configúrelos (vea el video al final) y presione el botón Atrás.
  • 8. Después de presionar el botón Reproducir en la parte superior derecha de la pantalla.
